## 【AI】

### Mistral 发布 Leanstral，面向 Lean 4 代码与形式化证明
3 月 16 日，Mistral 发布 Leanstral，并称这是一个面向 Lean 4 的开源代码 agent，可在形式化仓库里完成证明和实现任务；官方说明提到模型权重采用 Apache 2.0 许可，并提供免费 API 端点。  
**来源：** [Mistral](https://mistral.ai/news/leanstral) · [Hacker News](https://news.ycombinator.com/item?id=47404796)

### Hacker News 热帖讨论“Language Model Teams as Distributed Systems”论文
3 月 16 日，Hacker News 首页出现《Language Model Teams as Distrbuted Systems》，帖子指向一篇把多模型协作当作分布式系统来讨论的 arXiv 论文；截至抓取时，该帖记录为 64 points、28 条评论。  
**来源：** [arXiv](https://arxiv.org/abs/2603.12229) · [Hacker News](https://news.ycombinator.com/item?id=47401901)

### Launch HN 项目 Voygr 介绍面向 agents 和 AI 应用的地图 API
3 月 16 日，Launch HN 的《Voygr》介绍了一套面向 agents 和 AI 应用的 maps API，项目页称它把地点数据与新闻、文章、活动等网页上下文结合，用于判断商户是否仍在营业、是否更名或关闭；截至抓取时，该帖记录为 63 points、46 条评论。  
**来源：** [Hacker News](https://news.ycombinator.com/item?id=47401042) · [Voygr](https://voygr.tech/)

### NVIDIA 发布 Vera CPU，定位于 agentic AI 与强化学习基础设施
3 月 16 日，NVIDIA 在 GTC 宣布 Vera CPU，称其面向 agentic AI、强化学习、数据处理和 agent 推理场景；官方新闻稿写到，该产品号称比传统机架级 CPU 效率高 2 倍、速度快 50%，并提到阿里巴巴、字节跳动、Meta、Oracle Cloud Infrastructure 等合作部署方。  
**来源：** [NVIDIA Newsroom](https://nvidianews.nvidia.com/news/nvidia-launches-vera-cpu-purpose-built-for-agentic-ai) · [Hacker News](https://news.ycombinator.com/item?id=47404074)
